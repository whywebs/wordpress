Widget for the Automatic Backlinks link exchange network. Displays backlinks from the network and earns you backlinks in return.

== Description ==

Automatic Backlinks is a popular high quality link exchange platform. Your links are automatically placed on other members sites simply by displaying links on your own pages.

Members can restrict backlinks by topic, language, Majestic flow metrics and many more settings and can reject any backlinks they do not want.

Automatic Backlinks is the largest and oldest quality link exchange network on the web and is safe and highly effective.

== Installation ==

Once the zip file has been downloaded to your local computer:

1. From Plugins, click Add New, then Upload, select the previously downloaded zip file and click Install Now

2. Click the 'Activate' link for Automatic Backlinks Widget on your Plugins page.

3. Check the 'Settings" link and paste in your Automatic Backlinks account code 

4. Go to Appearance, Widgets and drag and drop the widget to wherever you want to show it on the Sidebar.

You can see how it all works on [http://www.automaticbacklinks.com/](http://www.automaticbacklinks.com/)
